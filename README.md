# maquina-ca-a-n-quel
Um algoritmo via Terminal de máquina de slot (caça-níquel), onde você começa com R$100 e pode ir alavancando seu dinheiro. Contém diversos elementos diferenciais, como a representação gráfica de quais elementos sairam no slot, contagem da maquina rolando, sistema de ganha e perca do dinheiro.
